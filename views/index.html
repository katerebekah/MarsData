MarsData


NASA's MAAS API gives access to weather data transmitted to Earth by the Mars Rover. This project uses this data and d3.js to make data visualizations about the weather on Mars. Data includes:

  • High Temperatures
  • Low Temperatures
  • Atmospheric Pressure
  • Sunrise and Sunset Time

See this project on GitHub.
